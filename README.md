ag-Grid Web Component
==============

This project contains the example of using ag-Grid Web Component 

See the [www.ag-grid.com](http://www.ag-grid.com) for an overview and full documentation.

#### Install with Bower
```sh
$ bower install 
```

#### Install with Npm
```sh
$ npm install 
```

#### To run the examples

#### Simple Grid
```sh
$ npm run simple
```

#### Rich Grid
```sh
$ npm run rich
```
